Parking Database Management System

School Server Edition - Apache + Python + SQLite

System Information:
This system provides two ways to manage your parking database:
  1. Static SQL Generator - Works on any web server, generates SQL commands
  2. Python CGI Application - Full database functionality with SQLite

SQL Generator Mode

Generate SQL commands for Oracle database without server-side execution

  • No server requirements
  • Works everywhere
  • Oracle SQL compatible
  • Copy & paste commands

Python CGI Mode

Full database application with SQLite backend

  • Add/edit/delete records

Setup Instructions:

  1. Run ./setup-school.sh to initialize the database
  2. Ensure CGI is enabled for your account (contact IT if needed)
  3. Access the application through your web browser

Troubleshooting:

  • 500 Internal Server Error: Check file permissions (755 for .py files)
  • CGI not executing: Verify .htaccess is being processed
  • Database errors: Ensure parking.db has write permissions (666)
  • Python not found: Check shebang line points to correct Python path
